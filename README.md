Smart To-Do Liste mit TinyLLM

Dies ist eine einfache To-Do-Listen-Anwendung in Python, die ich im Rahmen meiner Umschulung zur Fachinformatikerin für Anwendungsentwicklung entwickelt habe.

Das Projekt nutzt das kleine Sprachmodell TinyLlama, um aus freien Texteingaben automatisch strukturierte To-Do-Listen zu erstellen. Jede Aufgabe wird mit einer Priorität und einer Kategorie (Zuhause, Arbeit, Hobby, Sonstiges) versehen. Typische Eingangsfloskeln wie „Ich muss heute“ werden dabei automatisch entfernt.

Technologien:
Python
TinyLlama (LLM)
transformers / llama-cpp-python

Starten des Programms:
Das Programm wird mit dem Befehl python smart_todo.py gestartet. Aufgaben können durch Semikolon (;) getrennt eingegeben werden. Mit dem Befehl exit wird das Programm beendet.

Eine ausführliche Beschreibung des Projekts sowie Details zur Implementierung befinden sich in der Datei docs/Projektdokumentation.md.
